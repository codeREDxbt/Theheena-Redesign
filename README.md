# The Heena Redesign

Professional Image Consulting & Styling Portfolio for India's leading MSME-recognized Image Consultant, The Heena.

This project is a modern, responsive web application built with [Next.js](https://nextjs.org), designed to showcase curated style transformations, brand collaborations, media appearances, and executive coaching services.

## Features

- **Neo-Minimalist Design System**: A premium, editorial aesthetic using ivory and charcoal tones.
- **Dynamic Portfolio Gallery**: A responsive masonry layout showcasing 60+ curated style transformations and media appearances.
- **Scroll-to-Top Navigation**: Smooth scrolling UX enhancements across main pages.
- **Responsive Architecture**: Built with Tailwind CSS and Next.js App Router for optimal performance across all devices.
- **Optimized Assets**: Next.js image optimization for fast loading of high-resolution portfolio imagery.

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
